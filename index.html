COMPUTER VISION

  • Image Segmentation for Brain Tumor Detection in MRI Images

    I developed and deployed a deep learning-powered web application for brain tumor classification using the EfficientNetB3 model, achieving an accuracy of 99.54% on a dataset comprising pituitary, meningioma, glioma, and non-tumor cases. I integrated the model into a Flask-based web app, allowing users to upload MRI images for real-time predictions with confidence scores. The application is hosted on an AWS EC2 instance for live testing, showcasing my expertise in deep learning, model deployment, and cloud infrastructure.

    View Project Code

  • Football Analysis using YOLO

    This project focuses on detecting and tracking players, referees, and footballs in match footage using YOLO model. To enhance accuracy, I trained the model further on a custom football dataset, and used K-means clustering for team classification based on jersey colors. The system analyzes ball possession percentages and utilizes optical flow to measure camera movement, ensuring precise player tracking. I implemented perspective transformation to convert pixel-based movement into real-world distances, enabling accurate calculations of player speed and distance covered.

    View Project Code
